## Example

用于生成Kiwi测试Case的模版，可以在构建时选择要测试的类 

## Installation

打开终端输入sudo sh install-templates.sh

```
sudo chmod 755 install-templates.sh

sudo sh install-templates.sh 
```
运行完毕后会显示

```
Finished

```
